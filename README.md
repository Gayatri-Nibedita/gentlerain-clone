# Gentlerain AI – Frontend Clone

This project is a frontend clone inspired by https://gentlerain.ai  
Created as part of a Web Development assignment.

## Features
- Parallax scrolling effects
- Scroll-triggered animations
- Horizontal scrolling section
- Flip card hover animations
- Sticky navigation bar
- Fully responsive design

## Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
- GSAP
- AOS Animation Library

## Responsiveness
The website is fully responsive and optimized for mobile, tablet, and desktop devices.

## Performance
- Optimized animations
- Reduced motion on mobile
- Lightweight assets

## AI Usage Disclosure
AI tools (ChatGPT) were used for:
- Understanding animation logic
- Structuring code efficiently
- Debugging layout and responsiveness issues

All code was manually written, customized, and tested by me.

## Deployment
Deployed using Netlify.
